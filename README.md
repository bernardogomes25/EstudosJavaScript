# JavaScript & TypeScript — Do Zero

Repositório de estudos para aprender JavaScript e TypeScript do zero, cobrindo desde os fundamentos da linguagem até a construção de pequenos projetos práticos.

## Objetivo

O objetivo deste projeto é consolidar o aprendizado de JavaScript moderno e TypeScript de forma progressiva, passando por conceitos básicos até a aplicação deles em desafios reais de front-end.

## Estrutura do Projeto

- **`fundamentals/`** — Scripts com exercícios dos conceitos fundamentais do JavaScript (operadores, arrays, objetos, laços, datas, etc.)
- **`challenges/`** — Projetos práticos e desafios de front-end que aplicam os conceitos aprendidos
- **`new-content/`** — Conteúdo das aulas mais recentes, adicionado de forma contínua ao longo do curso
- **`template/`** — Template base com estrutura HTML/CSS/JS pronta para iniciar novos projetos rapidamente

## Conteúdo Abordado

### Fundamentos
- Operadores aritméticos, de comparação e lógicos
- Operadores condicionais (`if/else`, operador ternário)
- Laços de repetição (`for`, `while`)
- Arrays e seus métodos
- Objetos e propriedades
- Funções e escopo
- Datas com o objeto `Date`
- Geração de números aleatórios

### Desafios Práticos
- **BMI Calculator** — Calculadora de IMC com interface HTML/CSS
- **Calendar** — Calendário interativo gerado via JavaScript
- **HTML Manipulation** — Manipulação de elementos do DOM
- **Todo App** — Aplicação de lista de tarefas em Node.js

## Tecnologias

- JavaScript (ES6+)
- TypeScript *(em breve)*
- HTML5
- CSS3

## Como usar

Abra os arquivos `.html` dos desafios diretamente no navegador, ou execute os arquivos `.js` com Node.js:

```bash
node src/challenges/todo-app/index.js
```
